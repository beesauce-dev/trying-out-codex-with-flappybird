# Flappy Bird (Android)

This repository contains a simple Flappy Bird-style Android game written in Kotlin using a custom `SurfaceView` game loop.

## How to run

1. Open the project in Android Studio.
2. Sync Gradle.
3. Run the `app` configuration on an emulator or device.

## Gameplay

- Tap the screen to flap.
- Avoid the pipes.
- Tap after a game over to retry.
